Projects

Event Management System

Designed and developed an object-oriented Event Management System using Java to manage and organize events efficiently. The system allows users to create, view, update, and manage event details while maintaining structured data using CSV files as a lightweight database. The project demonstrates the application of object-oriented programming principles such as encapsulation, modular design, and class-based architecture, while also incorporating a graphical user interface using Java Swing/JavaFX for user interaction.

Automotive Crashworthiness Simulation (ANSYS LS-DYNA)

Performed a crashworthiness evaluation of an automotive frame using ANSYS LS-DYNA to study deformation behavior and energy absorption during impact scenarios. The project involved importing STL models, configuring simulation parameters, and analyzing structural response under crash conditions to understand vehicle safety performance.

AR Hand-Tracking Interactive Application

Developed an experimental augmented reality application that allows users to manipulate virtual objects using real-time hand gestures. The system integrates gesture recognition and interactive visual elements to create an intuitive interface for controlling digital objects through natural hand movements.

Human-Following Autonomous Robot

Developed a four-wheel autonomous robot using Arduino that can detect and follow human movement in real time. The system integrates ultrasonic and infrared sensors to track distance and direction, while motor drivers dynamically adjust the robot’s motion. This project demonstrates practical knowledge in embedded systems, sensor integration, and real-time control logic.

Vehicle-Mounted Wind Energy System

Designed a conceptual system to generate electrical energy using wind turbines mounted on moving vehicles. The project explores how airflow generated by vehicle motion can be harnessed to produce usable energy. It demonstrates understanding of renewable energy concepts, system design, and innovative approaches to energy harvesting.

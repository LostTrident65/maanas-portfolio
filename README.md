# Maanas Vishnu – Portfolio Website

This repository contains the source code for my personal portfolio website. The website is designed to present my profile, technical skills, and projects in a structured and visually engaging manner.

---

## Project Overview

The portfolio is a static web application built using HTML and CSS. It focuses on clean layout design, smooth user experience, and clear presentation of information. The site is structured into multiple sections that guide a visitor through my background, work, and contact details.

---

## Website Structure

The website is organized into the following main sections:

### 1. Landing / Hero Section

* Introduces the website with a visual background (video or image)
* Displays name, title, and a brief tagline
* Designed to create a strong first impression

### 2. About Section

* Provides a short introduction about me
* Highlights academic background and interests
* Gives context about my career direction

### 3. Skills Section

* Lists technical skills and tools
* May include categories such as programming, frameworks, and technologies
* Structured for quick readability

### 4. Projects Section

* Showcases selected projects
* Each project typically includes:

  * Title
  * Description
  * Image preview
* Demonstrates practical application of skills

### 5. Contact Section

* Provides ways to get in touch
* May include email, social links, or a contact form

---

## Folder Structure

The project follows a simple and clear directory structure:

```
maanas-portfolio-main/
│
├── index.html        # Main HTML file containing all sections
├── style.css         # Stylesheet for layout and design
│
├── images/           # Stores all image assets used in the website
│   ├── profile.jpg
│   ├── project images
│   └── other visuals
│
└── videos/           # Stores background and media videos
    ├── background videos
    └── section videos
```

---

## Code Structure

### HTML (index.html)

* Contains the complete structure of the website
* Divided into semantic sections (header, sections, footer)
* Each section corresponds to a part of the portfolio (about, skills, projects, etc.)

### CSS (style.css)

* Handles layout, colors, spacing, and responsiveness
* Uses modern layout techniques such as Flexbox and Grid
* Ensures the website adapts to different screen sizes

---

## Design Approach

* Minimal and clean UI to ensure readability
* Consistent spacing and alignment across sections
* Visual elements (images/videos) used to enhance engagement
* Focus on simplicity rather than heavy frameworks

---

## Usage

To run the website locally:

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in any web browser

No additional setup or dependencies are required.

---

## Deployment

The website can be deployed on any static hosting platform such as:

* GitHub Pages
* Vercel
